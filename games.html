_ ALS Level _

To the left is a video of my final level for my MI330 Level Design class. The goal of this level was to create a three part level: an infiltration stage, a stealth stage, and an escape stage. We were given an ALS template as well as blueprints and some prefabs to help us with creating the level. In this project I did not code anything, all code was done by Jeremy Bond, my professor at the time. I modeled most assets in the level of the landscape, building, cubicles, elevator, and the helicopter. I did not model the enemy or player characters, the collectibles(the cookie and the Rhino), or the bow. This was a multi week project.
