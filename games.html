_ ALS Level _

To the left is a video of my final level for my MI330 Level Design class. The goal of this level was to create a three part level: an infiltration stage, a stealth stage, and an escape stage. We were given an ALS template as well as blueprints and some prefabs to help us with creating the level. In this project I did not code anything, all code was done by Jeremy Bond, my professor at the time. I modeled most assets in the level of the landscape, building, cubicles, elevator, and the helicopter. I did not model the enemy or player characters, the collectibles(the cookie and the Rhino), or the bow. This was a multi week project. If you want to see my thoughts going through each stage weekly here is a link to a shortened ALS Level Design Journal

_ Pumpkin Panic _

To the Left is a trailer of a game I made in a group for my MI445 class project 2. My role was Lead designer I was in charge of coming up with ideas and helping the other members of the team implement them in a way that is friendly to the player. this game is a shop sim with a little rougelike aspect of an upgrade system using Tarot cards. All of the customers of the game are controlled by Chat GPT prompts that we mapped out by giving prompts on how to act, what to ask, and how it should respond when the picture of the carved pumpkin is sent to Chat GPT. This is a link to the game on Itch.ioPumpkin Panic this will give you more information about the team and the game, but in order to play you will need your own Chat GPT key. This is a link to the document that lays out what I did in more detailPumpkin Panic Design Journal.
